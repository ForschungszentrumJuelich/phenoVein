phenoVein is a software tool designed for the segmentation and analysis of leaf venation images.

For license conditions, see License.txt

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Installation of phenoVein:

 - download MeVisLab 2.6.1 from www.mevislab.de
 - install MeVisLab
 - extract phenoVein to a local directory
 - make phenoVein known to MeVisLab, in MeVisLab go to
    -> Edit -> Preferences -> Packages -> Add Existing User Packages -> choose phenoVein directory

Start phenoVein in MeVisLab from the menu bar: Applications -> phenoVein
